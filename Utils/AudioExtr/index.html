🎵 Утилита: Извлечение Аудио (AudioExtr)

💡 Что делает эта утилита?

AudioExtr — это удобный скрипт для пакетного извлечения звука из видеофайлов. Если у вас есть папка с клипами, концертами или интервью, и вы хотите перевести их в аудиоформат для прослушивания (например, в MP3, WAV или FLAC), этот скрипт сделает всё автоматически.

⚙️ Как работает эта утилита?

  • Пакетная обработка: Скрипт находит все видео в папке и по очереди извлекает из них звуковые дорожки.
  • Выбор формата: Вы можете выбрать формат, в который нужно сохранить аудио (MP3, WAV, FLAC, AAC, M4A, OGG). Качество настраивается оптимально для каждого формата.
  • Сохранение в подпапку: Все готовые треки будут аккуратно сложены в новую папку (например, `Audio_MP3`), чтобы не смешиваться с оригиналами.
  • Не требует установки тяжелых программ: Используется движок ffmpeg прямо из скрипта.

💻 Как запустить скрипт?

  1. Скачайте файл утилиты по кнопке «СКАЧАТЬ (.PS1)».
  2. Переместите скачанный файл Extr_AUDIO.ps1 в папку, где лежат ваши видеофайлы.
  3. Кликните по файлу правой кнопкой мыши и выберите пункт «Выполнить с помощью PowerShell».
  4. Следуйте инструкциям на экране (нужно будет ввести цифру желаемого аудиоформата).
